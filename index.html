CPS530 Lab 10 Tailwind CSS

1. Framework which I chosen

For problem2, I chose the Tailwind CSS framework. Tailwind is a utility-first CSS framework that eliminates the need to write traditional CSS and requires no installation, offering greater flexibility.

2. How to install

use code in head:
<script src="https://cdn.tailwindcss.com"></script>
After adding the code, all Tailwind CSS classes can be used.

3. How to build page

Create a GitHub repository, add the file index.html, and host it using GitHub Pages.

4. Difficulties I faced

  • I have doubts about choosing Tailwind CSS and whether it's efficient.
  • After uploading to GitHub Pages, it fails to display.

5. Solve the problem

  • I directly called Tailwind CSS and found it extremely convenient.
  • Following the GitHub Pages manual, I renamed the file to index.html and changed the main/root directory.
